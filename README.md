## 文章的格式
- 文章使用markdown格式。具体参照[http://md.barretlee.com/](http://md.barretlee.com/)。
- 文章所用的图片不能有水印，新建一个images文件用来放图片。
- 文章需要有封面头图，展示在微信分享的缩略图。
- 作者需要校验错别字，使用的markdown语法不应太复杂。
